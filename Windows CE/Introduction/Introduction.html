Working with the SDK
The Handpoint WinCE SDK is compatible with .NET Compact Framework Version 3.5.

1. Download the SDK.
2. Create a new project in Visual Studio 2008 using .NET Compact Framework Version 3.5 with the target platform as Windows CE.
3. Add "hapi-dotnet-2008.dll", "hapi-dotnet-wince.dll" and "InTheHand.Net.Personal" libraries as references to your project.
4. Follow the getting started guide, to get you up and running or dive into the documentation.
